Code Explanation

The sun is represented by a central sphere, and planets orbit around it in 3D space. Each planet's position is calculated using the cosine and sine of an angle, which is incremented over time to create the orbital motion.

The JavaScript code starts by declaring global variables: sun to store the sun object and planets to store multiple planet objects in an array.

Sine and cosine are trigonometric functions that relate the angles of a right triangle to the lengths of its sides. For an angle θ:

  • cos(θ) is the ratio of the adjacent side to the hypotenuse.
  • sin(θ) is the ratio of the opposite side to the hypotenuse.
In this context, they are used to calculate the x and y positions of the planets in their circular orbits.

An array is a data structure that can hold multiple values at once. In JavaScript, arrays are used to store lists of items, such as the planets array in this example. Each item in an array can be accessed by its index, which starts at 0.

Explanation of the 3D Solar System Code

Global Variables:

  • sun: Variable to store the sun object.
  • planets: Array to store multiple planet objects.

Setup Function:

  • createCanvas(800, 800, WEBGL): Creates a 3D canvas of size 800x800 pixels.
  • sun = new Planet(50, 0, 0, color(255, 204, 0)): Initializes the sun with a radius of 50, at the center of the canvas, and a yellow color.
  • planets.push(new Planet(20, 150, 0.02, color(100, 100, 255))): Adds a blue planet with a radius of 20, a distance of 150 from the sun, and a speed of 0.02.
  • planets.push(new Planet(30, 300, 0.01, color(255, 100, 100)): Adds a red planet with a radius of 30, a distance of 300 from the sun, and a speed of 0.01.

Draw Function:

  • background(0): Sets the background color to black.
  • directionalLight(255, 255, 255, 1, 1, -1): Sets a directional light.
  • ambientLight(50): Adds ambient light to the scene.
  • rotateY(frameCount * 0.01): Rotates the entire scene for a better view.
  • sun.show(): Displays the sun.
  • for (let planet of planets) { planet.orbit(); planet.show(); }: Loops through each planet, updates its position, and displays it.

Planet Class:

  • constructor(r, d, s, c): Initializes the planet's properties: radius, distance from the sun, initial angle for the orbit, speed of the orbit, and color.
  • orbit(): Updates the planet's position by increasing the angle by the speed.
  • show():
    • push() and pop(): Saves and restores the current drawing style and transformations.
    • fill(this.c): Sets the fill color to the planet's color.
    • noStroke(): Disables the stroke.
    • translate(x, y): Moves to the planet's position.
    • sphere(this.r): Draws the planet as a sphere.

    
// Variable to store the sun object
let sun;

// Array to store multiple planet objects
let planets = [];

// The setup function runs once when the program starts
function setup() {
  // Create a 3D canvas of size 800x800 pixels
  createCanvas(800, 800, WEBGL);

  // Initialize the sun with a radius of 50, at the center of the canvas, and a yellow color
  sun = new Planet(50, 0, 0, color(255, 204, 0));

  // Initialize two planets with different properties
  planets.push(new Planet(20, 150, 0.02, color(100, 100, 255))); // Blue planet
  planets.push(new Planet(30, 300, 0.01, color(255, 100, 100))); // Red planet
}

// The draw function runs continuously in a loop
function draw() {
  // Set the background color to black
  background(0);

  // Set lights for the 3D environment
  directionalLight(255, 255, 255, 1, 1, -1);
  ambientLight(50);

  // Rotate the entire scene for better view
  rotateY(frameCount * 0.01);

  // Display the sun
  sun.show();

  // Loop through each planet in the planets array
  for (let planet of planets) {
    // Update the planet's position
    planet.orbit();
    // Display the planet
    planet.show();
  }
}

// Class to represent a planet
class Planet {
  // Constructor to initialize the planet's properties
  constructor(r, d, s, c) {
    this.r = r; // Radius of the planet
    this.d = d; // Distance from the sun
    this.angle = random(TWO_PI); // Initial angle for the orbit
    this.speed = s; // Speed of the orbit
    this.c = c; // Color of the planet
  }

  // Method to update the planet's position
  orbit() {
    this.angle += this.speed; // Increase the angle by the speed to create orbital motion
  }

  // Method to display the planet
  show() {
    push();
    fill(this.c); // Set the fill color to the planet's color
    noStroke(); // Disable the stroke
    let x = this.d * cos(this.angle); // Calculate the x position using cosine
    let y = this.d * sin(this.angle); // Calculate the y position using sine
    translate(x, y); // Move to the planet's position
    sphere(this.r); // Draw the planet as a sphere
    pop();
  }
}
